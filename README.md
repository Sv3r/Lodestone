# Lodestonelib-fabric

## FAQs
### How to install?
Add this in the gradle `repositories` block:
```groovy
maven { url 'https://jitpack.io' }
```
...and this in the dependencies block:
```groovy
modImplementation 'com.github.cph101:Lodestonelib-fabric:v0.0.2-1.20.1'
```
